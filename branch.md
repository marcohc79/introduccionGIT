
# Branch
Permite **ver, crear o eliminar ramas** en el repositorio. Este comando es Ãºtil para gestionar diferentes versiones del proyecto sin afectar la rama principal.
**Listar todas las ramas ejemplo:**
```bash

git branch

```
**Crear una nueva rama ejemplo:**
```bash

git branch [nombre-rama]

```
**Eliminar una rama ejemplo:**
```bash

git branch -d [nombre-rama]

```
